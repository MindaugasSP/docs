## Web Scraping API Amazon Target

Amazon can be scraped using multiple different targets, including the **Amazon search**, **direct URL**, or more specific targets like **Amazon seller**, **Amazon product**, etc.

In each respective target template page, you will find:

* Whether it can be parsed with `JSON`;
* The required parameters;
* Ready-to-use code examples for `cURL`, `Node`, and `Python`, where you only need to change the username and password for your Web Scraping API user for the code to work.
