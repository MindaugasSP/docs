# 身份验证方法

住宅代理身份验证方法

我们支持通过用户名和密码或基于 IP 的白名单进行住宅代理身份验证。

您可以在仪表板的 [**住宅 → 住宅 → 身份验证**](https://dashboard.decodo.com/residential-proxies/authentication) 部分管理**代理用户**和**白名单 IP**。

* 购买计划或订阅后，代理用户名和密码将自动在身份验证部分生成。
* 白名单 IP 插槽的默认数量为 **10** 个。如需更多信息，请[**联系**](https://direct.lc.chat/12092754/1)我们的客户支持。

> 📘 仪表板访问
>
> 如果您无法登录仪表板，请访问[**重置密码**](https://dashboard.decodo.com/password-reset)页面请求更改密码。

***

## 用户名:密码身份验证

### 管理代理用户

* 您可以通过点击身份验证部分的**用户**选项卡来管理您的**代理用户**。
* 您可以通过点击右侧的**购买用户**来**购买**额外的用户。

<br />

### 编辑代理用户

* 要**编辑**代理用户，请点击**三个点**，然后点击**编辑用户**。右侧将打开一个菜单。
* 或者，您可以直接在[**代理设置**](https://dashboard.decodo.com/residential-proxies/proxy-setup)部分**添加**或**编辑**代理用户、生成**密码**并更改其流量限制。


<br />

### 密码生成

* 要**更改**代理用户的**密码**，请点击密码旁边的**生成密码**按钮。
* 要显示或隐藏密码，请点击**眼睛**图标。
* 您可以通过点击用户名和密码来**复制**它们。

### 流量限制管理

1. 要为特定代理用户**添加**、**编辑**或**删除**流量限制，请点击**添加流量限制**。
2. 输入流量限制并点击对勾按钮以添加或编辑限制。点击 **X** 取消。
3. 点击**编辑**进行更改，或点击**垃圾桶**图标**删除**流量限制。
4. 点击**保存**

### 删除代理用户

* 要**删除**代理用户，请点击**删除用户**，然后点击**确认**。

<br />

### 添加额外的代理用户

1. 要**创建**新的代理用户名和密码，请点击 **+ 添加用户**。右侧将打开一个菜单。

   * 此选项需要有空位才能显示。请检查**总用户数**。
2. 输入您想要的用户名并点击 **+ 添加**。密码将自动生成。
   * 用户名必须为 6 到 64 个字符，仅限字母和数字，不能有空格。

> 🚧 407 错误 / 需要代理身份验证
>
> 如果您无法使用代理用户凭据进行身份验证，请尝试生成新密码。

***

## 白名单 IP 身份验证

### 管理白名单 IP

1. 您可以通过点击身份验证部分的**白名单 IP** 选项卡来管理您的**白名单 IP**。
   * 默认情况下列表为空。
2. 要向白名单**添加**新的 **IP 地址**，请点击 **+ 添加 IP**。右侧将打开一个菜单。
   * 要**购买**超过 10 个白名单 IP 插槽，请通过点击右下角的聊天小部件或使用此[**链接**](https://direct.lc.chat/12092754/1)联系我们的 24/7 客户支持。



<br />

3. 输入您想要的 **IP 地址**，然后点击 **+ 添加**将其加入白名单。
   * 您只能添加 **IPv4** 格式（xx.xx.xx.xx）的 IP。不支持 **IPv6** 格式。
   * 确保您添加的 IP 是您的，只有您可以访问，并且您在添加时没有使用代理或 VPN 服务。


<br />

### 编辑白名单 IP

* 要**编辑**白名单 IP，请点击**三个点**，然后点击**编辑 IP**。右侧将打开一个菜单。
* 或者，您可以直接在[**代理设置**](https://dashboard.decodo.com/residential-proxies/proxy-setup)部分**添加**或**编辑**白名单 IP。


<br />

### 命名白名单 IP

* 要**命名**您的白名单 IP，请点击**编辑**并输入您想要的名称。点击**保存**完成。
* 将输入字段留空以删除名称。

### 删除白名单 IP

* 要**删除**白名单 IP，请点击**删除白名单 IP**，然后点击**确认**。
