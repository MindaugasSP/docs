# 目标模板

网页抓取 API 目标

WEB API 高级版计划包含一系列目标模板，旨在提高抓取效率。

1. 专业的解锁策略 – 确保高成功率。
2. 目标特定参数 – 针对特定目标优化的设置。
3. 专用解析器 – 为支持的目标提供预构建的解析解决方案。

<Callout icon="📘" theme="info">
  此功能适用于 [**Web API 高级版计划**](https://dashboard.decodo.com/register?page=scrapers/pricing??plan=907782)
</Callout>
