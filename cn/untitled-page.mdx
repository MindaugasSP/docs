# 身份验证

## 身份验证方法

### 住宅代理身份验证方法

我们支持通过用户名和密码或基于 IP 的白名单进行住宅代理身份验证。

您可以在仪表板的**住宅 → 住宅 → 身份验证**部分管理代理用户和白名单 IP。

购买计划或订阅后，代理用户名和密码将自动在身份验证部分生成。

白名单 IP 插槽的默认数量为 **10**。请联系我们的客户支持以获取更多信息。

> 📘 仪表板访问
>
> 如果您无法登录仪表板，请访问[**重置密码**](https://dashboard.decodo.com/reset-password)页面请求更改密码。

<br />

## 管理代理用户

您可以通过点击身份验证部分中的**用户**选项卡来管理代理用户。

您可以通过点击右侧的**购买用户**来购买额外的用户。

目前，您无法重命名代理用户。如果需要，您可以创建一个新用户或通过删除来重新创建代理用户。
