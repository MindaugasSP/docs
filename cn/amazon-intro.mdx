## 网页抓取 API Amazon 目标

可以使用多种不同的目标抓取 Amazon，包括 **Amazon 搜索**、**直接 URL** 或更具体的目标，如 **Amazon 卖家**、**Amazon 产品**等。

在每个相应的目标模板页面中，您将找到：

* 是否可以使用 `JSON` 解析；
* 所需参数；
* 适用于 `cURL`、`Node` 和 `Python` 的即用代码示例，您只需更改网页抓取 API 用户的用户名和密码即可使代码正常工作。

<Callout icon="📘" theme="info">
  Amazon 目标模板适用于 [**Web API 高级版计划**](https://dashboard.decodo.com/register?page=scrapers/pricing?plan=907766)。
</Callout>
